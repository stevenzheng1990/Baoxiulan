# Lottie 动画文件

机构设施页面的康复项目支持 Lottie 动画（After Effects 导出）。
将下载好的 `.lottie` 或 `.json` 文件按以下文件名放在此目录即可，系统会自动启用：

| 文件名 | 对应项目 | 推荐 LottieFiles 关键词 |
|---|---|---|
| `early.lottie`     | 早期综合训练 | baby, infant, newborn, lullaby |
| `inclusion.lottie` | 融合教育     | children, classroom, kindergarten |
| `speech.lottie`    | 言语认知训练 | speech, conversation, learning |
| `motor.lottie`     | 运动训练     | exercise, balance, fitness kids |
| `behavior.lottie`  | 行为矫正训练 | reward, task, autism therapy |
| `sensory.lottie`   | 感觉统合训练 | swing, sensory, play |
| `ot.lottie`        | 作业训练     | hand, fine motor, blocks |
| `tuina.lottie`     | 小儿推拿     | massage, traditional, hands |

## 推荐获取方式

1. https://lottiefiles.com 搜索关键词，挑选风格统一的免费动画
2. 优先选择「medical / line / minimal」风格，与品牌蓝+烫金调性匹配
3. 下载 `.lottie` 格式（dotLottie，单文件包含所有资源），重命名后放入本目录

## 若不放文件会怎样？

每个项目都有精心设计的 SVG 场景兜底，包含多元素的具象插图 + 完整动画编排。
放入 Lottie 后自动替换为 Lottie 播放器；移除后回到 SVG 兜底。
